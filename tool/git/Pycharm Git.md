# Pycharm和Git联动

### 1. 删除Git

##### 1.1 取消Git关联

​      settings --> Version Control --> VCS "None"

##### 1.2 删除.Git

1. 用资源管理器打开：Open In  --> Directory Path

2. 用Git bash打开项目

   ```
   find . -name ".git" | xargs rm -Rf
   ```
